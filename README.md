# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. This project showcases my skills, experience, and projects in a clean and visually appealing layout.

## Features

- **Responsive Design:** Fully responsive layout that works seamlessly on mobile, tablet, and desktop devices.
- **Dark/Light Mode:** Toggle between dark and light themes for better readability.
- **Smooth Scrolling:** Smooth navigation between different sections of the page.
- **Interactive Elements:** Hover effects, animations, and a mobile-friendly hamburger menu.
- **Contact Form:** A styled contact form (frontend only).
- **Downloadable Resume:** Direct link to download my resume.

## Technologies Used

- **HTML5:** Semantic markup for structure.
- **CSS3:** Custom styling, Flexbox, Grid, and CSS Variables for theming.
- **JavaScript (ES6+):** Logic for the theme toggle, mobile menu, and scroll animations.

## Prerequisites

To run this project locally, you need:

- A modern web browser (Chrome, Firefox, Edge, Safari).
- A code editor like [VS Code](https://code.visualstudio.com/) (optional, for editing).

Follow these steps to get the project up and running on your local machine:

### Option 2: Using VS Code Live Server (Recommended for Development)

1.  **Clone or Download the Repository:**
    *   If you have Git installed:
        ```bash
        git clone https://github.com/your-username/your-repo-name.git
        ```
    *   Or download the ZIP file and extract it.

2.  **Navigate to the Project Folder:**
    Open the folder where you extracted the files.

3.  **Open the Project in VS Code:**
    Launch Visual Studio Code and open the project folder.

4.  **Install "Live Server" Extension:**
    *   Go to the Extensions view (Ctrl+Shift+X).
    *   Search for "Live Server" by Ritwick Dey.
    *   Click **Install**.

5.  **Start the Server:**
    *   Open `index.html` in the editor.
    *   Right-click anywhere in the file and select **"Open with Live Server"**.
    *   Alternatively, click the "Go Live" button at the bottom right corner of the VS Code window.

6.  **View the Site:**
    The browser will automatically open (usually at `http://127.0.0.1:5500/index.html`), and you can see the running project.
